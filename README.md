#  Sentence generator
A simple (and very stupid) sentence generator

<div align="left">

# Building
### 1. Clone this repo

```ps
git clone https://github.com/agaqmk/Sentence-generator.git && cd ./Sentence-generator
```
### 2. Install dependencies

```ps 
npm i
```

### 3. Install pkg

```ps 
npm i -g pkg
```

### 4. Compile with pkg

```ps
pkg index.js
```
